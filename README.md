# One Nevada Credit Union Landing Page

This project is a React + Vite app styled with Tailwind CSS. It includes a landing page inspired by the attached One Nevada Credit Union homepage and is ready for local development.

## Available Scripts

- `npm install` — install dependencies
- `npm run dev` — start the development server
- `npm run build` — build the production app
- `npm run preview` — preview the built site locally

## Project setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

The app runs at `http://localhost:5173` by default.
